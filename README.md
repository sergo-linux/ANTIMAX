<img src="https://i.imgur.com/3SSmuJk.png" width=200>

## Как работает?
1. Использует `.ipa` с обычным черным экраном, в который инжектнут `.dylib` который вызывает "ошибку"
2. При открытии приложения срабатывает `.dylib`
3. При нажатии `ОК` приложения принудительно закрывается
